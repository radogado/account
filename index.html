About

A fintech dashboard template built on web standards and native HTML, which makes it fully accessible, and compliant. The page consists of sensible landmarks like "aside", "header", "nav" and "main", which help screen readers describe it properly. It uses CSS for styling, animation, and unbreakable layout. A minimal amount of JavaScript is used for enhancement.

The menu supports both "a" and "button" elements as navigation items. On menu shrink, the logo transitions smoothly to an initial letter version. Its dimensions are defined by the theme, which also specifies the accent colour and supports dark mode. The user profile pop-up uses the standard "details" element for native disclosure functionality. The dynamic translation uses a JSON file as dictionary and data attributes as bridge between the app and the UI.

This template achieves perfect score in the most important performance/accessibility benchmarks like Lighthouse, Wave and axe®.
